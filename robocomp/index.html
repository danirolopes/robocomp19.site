Quero Saber Mais!

O Projeto

Dividiu-se a turma em grupos de 3 integrantes, de forma que cada um dos integrantes tenha um papel fundamental no desenvolvimento do projeto, que são descritos abaixo.

Hardware

Construção da base do robô, além de um circuito de testes e uma base bara o microcontrolador através do software Eagle.

Mais Sobre o Hardware!

Microcontrolador

Desenvolver um programa com o software MpLab, que será executado em um microcontrolador 16F648A, que receberá comandos do App via módulo Bluetooth e controlará os motores do robô.

Mais Sobre o Microcontrolador!

Software

Desenvolver um App através do AppInventor do MIT, de forma que ele possa ser usado de controle do robô, recebendo comandos via interface com usuário e transmitindo via Bluetooth para o microcontrolador.

Mais Sobre o App!

Como Fizemos

Veja como desenvolvemos o projeto, quais foram nossos principais acertos e erros e qual foi o resultado!

Como todo projeto de engenharia, passamos por muitos problemas que não foram previstos no começo!
Portanto, nosso projeto não conta apenas com um manual de como fazer, mas também com um diário de bordo. Assim, mostramos como foi o processo de construção, quais erros cometemos (podem ser os mesmos que você está tendo!) e todos os percalços de conduzir um projeto do mundo real.
